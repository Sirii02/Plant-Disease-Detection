About Our Project

This project focuses on detecting plant leaf diseases by training a deep learning model using convolutional neural networks (CNN). We first trained the model on a dataset containing healthy and diseased leaf images. The trained model is then used to predict the health status of new uploaded images through a user-friendly web interface. The website includes key features such as Login, Register, Contact, About, Predict, and Result pages for smooth user interaction. The core algorithm is built using a Sequential model structure with convolutional layers for feature extraction and dense layers for classification. The model architecture includes Conv2D layers with 32 and 16 filters, MaxPooling layers, and a final dense layer with softmax activation to predict three classes. Once logged in, users can upload leaf images to instantly receive prediction results. This system provides accurate, fast, and accessible plant disease diagnosis for users through a well-designed web platform.
